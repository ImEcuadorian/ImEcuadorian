# 🔎 Featured Project — UPS Parking (Smart Parking Management)

**Role:** Fullstack / Backend Lead  
**Period:** Jun 2025 — Present  

**Description**

A smart parking management platform designed to optimize parking space allocation, reservations, and real-time monitoring within institutional environments.

**Technical Responsibilities & Contributions**

- Designed a **microservices-based architecture** separating reservation, availability, authentication, and telemetry services
- Implemented backend services using **Spring Boot microservices** with **Spring Cloud Gateway**
- Built the mobile application using **React Native (Expo)** for reservations, notifications, and parking map visualization
- Designed persistence layer using **PostgreSQL**, with caching and analytics using **Redis**
- Containerized services with **Docker** and implemented reproducible **CI/CD pipelines using GitHub Actions**
- Integrated **AI/NLP modules** for feedback analysis and semantic search capabilities
- Implemented **observability tooling** for logs, metrics, and tracing (Prometheus + Jaeger ready)

**Why it matters**

Improves operational efficiency in campus environments and reduces parking search time, improving overall user experience and logistics.

**Tech Stack**

React Native · Spring Boot (Microservices) · Spring Cloud Gateway · PostgreSQL · Redis · Docker · GitHub Actions · NLP Integration
